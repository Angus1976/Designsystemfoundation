# 问视间 (WINS) Logo 资源包

## 🎯 设计理念

**核心概念**："Wi" 波浪形 = 数据向好 = 智赢未来

- **"W" 字母**：前4个柱子倾斜形成标准 "W" 形状
- **"i" 字母**：第5个柱子直立 + 红色圆点形成 "i"
- **"Wi" 组合**：代表 "Wins" 的精简形式
- **波浪上升**：5柱形成向好的波浪曲线
- **WINS (问视间)**：中文"问视间"与英文"Wins"谐音
- **渐变配色**：蓝色 (#1890FF) 到绿色 (#52C41A)

## 🎨 设计元素详解

### 核心元素："Wi" 波浪形

```
视觉呈现：

         ●  ← 红点（i的点）
         █  ← 第5柱 - 直立的"i"
        ╱ ╲  
       █   █  ← 第3-4柱 - W的右侧
      ╱     ╲
     █       ╲  ← 第2柱 - W的谷底（最低）
    ╱         ╲
   █           ← 第1柱 - W的左侧
  ────────────────
      "Wi"
```

### 5柱详解

| 柱子 | 形态 | 高度 | 作用 | 象征意义 |
|------|------|------|------|----------|
| 第1柱 | 左倾斜↗ | 中等 | W左侧 | 数据起点 |
| 第2柱 | 右倾斜↖ | 最低 | W谷底 | 波动低谷 |
| 第3柱 | 左倾斜↗ | 中高 | W右侧 | 数据回升 |
| 第4柱 | 右倾斜↖ | 高 | W结束 | 持续上升 |
| 第5柱 | 直立 ｜ | 最高 | i柱体 | 数据巅峰 |
| 红点 | ● | - | i的点 | 智能洞察 ⭐ |

### 视觉元素（7个）

1. **前4个倾斜柱** - 形成 "W" 字母
   - 交替倾斜（左右左右）
   - 蓝绿渐变
   - 高低起伏

2. **第5个直立柱** - 形成 "i" 的主体
   - 完全垂直
   - 最高的柱子
   - 象征成功巅峰

3. **红色圆点** - "i" 的点 ⭐ 核心
   - 鲜明的红色 (#FF4D4F)
   - 位于第5柱顶部
   - 象征智能洞察、关键点

4. **白色数据点** - 5个关键节点
   - 每个柱子的关键位置
   - 标记数据节点
   - 连接成波浪

5. **波浪趋势线** - 向上的曲线
   - 连接5个数据点
   - 流畅的波浪形
   - 整体向上（数据向好）

6. **成功箭头** - 向上突破
   - 绿色箭头 ↑
   - 第5柱顶部
   - 强调上升趋势

7. **背景光晕** - 氛围烘托
   - 淡蓝绿渐变
   - 圆形背景
   - 低调衬托

```
波浪曲线 = 数据发展趋势
```

- 确保红点清晰可见

### 颜色使用

**适用背景**：
- ✅ 白色背景（最佳）
- ✅ 浅色背景（很好）
- ✅ 深色背景（良好，红点更醒目）
- ✅ 渐变背景（适用）

**禁止操作** ❌：
- ❌ 不要改变颜色（特别是红点）
- ❌ 不要拉伸变形
- ❌ 不要旋转 Logo（会破坏 Wi 形）
- ❌ 不要分离元素
- ❌ 不要移除红点（这是核心）
- ❌ 不要改变柱子倾斜角度

**推荐操作** ✅：
- ✅ 保持原始宽高比
- ✅ 等比例缩放
- ✅ 使用纯色背景
