## 🤖 AI 标注使用指南

### 第一步：进入 AI 标注页面

```
访问路径: /ai-annotation
或点击: 侧边栏 → AI 能力 → AI 标注
```

### 第二步：了解三大功能模块

#### 1️⃣ 任务列表（默认页面）

**功能**：
- 查看所有标注任务
- 监控任务进度和状态
- 查看准确率统计
- 导出标注结果

**操作指南**：
```
1. 查看任务 → 点击表格行查看详情
2. 搜索任务 → 使用顶部搜索框
3. 导出数据 → 点击"导出"按钮
4. 操作任务 → 点击"⋮"菜单 → 选择操作
```

**状态说明**：
- 🔵 **待处理** - 任务已创建，等待开始
- 🔵 **运行中** - 正在执行标注
- 🟢 **已完成** - 标注完成
- 🔴 **失败** - 执行出错

#### 2️⃣ 模型管理

**功能**：
- 查看可用 AI 模型
- 启用/禁用模型
- 查看模型准确率
- 配置模型参数

**操作指南**：
```
1. 切换标签 → 点击"模型管理"
2. 查看模型 → 浏览 6 个模型卡片
3. 启用模型 → 切换右上角开关
4. 配置模型 → 点击"配置"按钮
```

**模型类型**：
- 🟣 **NER** - 命名实体识别
- 🔵 **分类** - 文本分类
- 🟢 **分割** - 图像分割
- 🟠 **检测** - 目标检测

#### 3️⃣ 标注模板

**功能**：
- 使用预设模板
- 查看模板详情
- 快速创建任务

**操作指南**：
```
1. 切换标签 → 点击"标注模板"
2. 浏览模板 → 查看 6 个模板卡片
3. 查看详情 → 点击模板卡片
4. 使用模板 → 点击"使用"按钮
```

**模板类型**：
- 🏷️ 命名实体识别模板
- 📊 文本分类模板
- 🖼️ 图像语义分割模板
- 😊 情感分析模板
- 🔗 关系抽取模板
- 🎯 目标检测模板

### 第三步：创建第一个标注任务

```
1. 点击右上角"新建标注任务"按钮
2. 选择标注类型（NER/分类/分割/检测）
3. 选择使用的 AI 模型
4. （可选）选择标注模板
5. 上传数据文件
6. 配置标注参数
7. 点击"开始标注"
```

### 快捷操作

| 操作 | 快捷方式 |
|------|---------|
| 新建任务 | 点击右上角蓝色按钮 |
| 搜索任务 | 使用搜索框 |
| 切换标签 | 点击标签页导航 |
| 查看详情 | 点击表格行 |
| 快速操作 | 点击行尾"⋮"菜单 |

## 🎨 数据增强使用指南

### 第一步：进入数据增强页面

```
访问路径: /augmentation
或点击: 侧边栏 → AI 能力 → 数据增强
```

### 第二步：了解三大功能模块

#### 1️⃣ 概览（默认页面）

**功能**：
- 查看统计数据
- 监控增强任务
- 分析成功率

**操作指南**：
```
1. 查看统计 → 4 个统计卡片
   - 原始样本数量
   - 增强样本数量
   - 待处理任务
   - 成功率趋势
2. 查看任务 → 最近任务表格
3. 操作任务 → 点击"⋮"菜单
```

**统计卡片**：
- 🔵 **原始样本** - 已上传的原始数据
- 🟢 **增强样本** - 已生成的增强数据
- 🟠 **待处理** - 队列中的任务
- 🟣 **成功率** - 任务成功百分比

#### 2️⃣ 样本管理

**功能**：
- 查看所有样本
- 区分原始/增强
- 批量上传
- 导出样本

**操作指南**：
```
1. 切换标签 → 点击"样本管理"
2. 浏览样本 → 网格布局展示
3. 查看详情 → 点击样本卡片
4. 批量上传 → 点击"批量上传"按钮
5. 导出样本 → 点击"导出"按钮
```

**样本标识**：
- 🔵 **原始** - 蓝色标签
- 🟣 **增强** - 紫色标签

**样本状态**：
- ⚪ **待处理** - 灰色徽章
- 🔵 **处理中** - 蓝色徽章
- 🟢 **完成** - 绿色徽章

#### 3️⃣ 增强配置

**功能**：
- 配置增强方法
- 调整增强强度
- 保存配置方案

**操作指南**：
```
1. 切换标签 → 点击"增强配置"
2. 启用方法 → 切换方法开关
3. 调整强度 → 拖动滑块（0-100%）
4. 查看摘要 → 底部统计面板
5. 保存配置 → 点击"保存配置"按钮
```

**6 种增强方法**：

| 方法 | 图标 | 说明 | 默认强度 |
|------|------|------|---------|
| 🔄 **旋转变换** | RotateCw | 0-360度随机旋转 | 45% |
| ↔️ **水平翻转** | FlipHorizontal | 水平镜像翻转 | 100% |
| ✂️ **裁剪缩放** | Crop | 随机裁剪和缩放 | 30% |
| 🎨 **亮度调整** | Palette | 亮度随机变化±30% | 50% |
| 📊 **对比度** | Sliders | 对比度调整 | 40% |
| 🔍 **缩放变换** | ZoomIn | 0.8-1.2倍缩放 | 60% |

### 第三步：创建第一个增强任务

```
1. 点击右上角"新建增强任务"按钮
2. 上传原始样本数据
3. 切换到"增强配置"标签
4. 选择并配置增强方法
5. 设置每个方法的强度
6. 保存配置
7. 返回"概览"标签
8. 点击"开始增强"
```

**推荐配置方案**：

## 💡 使用场景示例

### 场景 1：批量标注商品图片

**需求**：标注 1000 张商品图片的类别

**操作流程**：
```
1. 访问 /ai-annotation
2. 点击"新建标注任务"
3. 选择"图像分类"类型
4. 选择 "YOLOv8-v1.0" 模型
5. 上传 1000 张图片
6. 开始自动标注
7. 在"任务列表"查看进度
8. 完成后导出结果
```

**预计时间**：10-20 分钟（取决于图片大小）

### 场景 2：扩充训练数据集

**需求**：将 500 张图片扩充到 2000 张

**操作流程**：
```
1. 访问 /augmentation
2. 点击"新建增强任务"
3. 上传 500 张原始图片
4. 切换到"增强配置"
5. 启用所有 6 种增强方法
6. 设置增强倍数为 4
7. 开始增强任务
8. 在"概览"查看进度
9. 完成后导出 2000 张图片
```

**预计时间**：5-10 分钟

### 场景 3：实体识别任务

**需求**：识别文本中的人名、地名、机构名

**操作流程**：
```
1. 访问 /ai-annotation
2. 切换到"标注模板"
3. 选择"命名实体识别模板"🏷️
4. 点击"使用"
5. 上传文本文件
6. 自动应用 BERT-NER 模型
7. 开始标注
8. 导出标注结果（JSON 格式）
```

**预计时间**：根据文本量而定

## ❓ 常见问题

### Q1: 如何提高标注准确率？

**A**: 
1. 选择适合的 AI 模型
2. 使用高质量的训练数据
3. 调整模型参数
4. 人工审核标注结果

### Q2: 增强任务失败怎么办？

**A**:
1. 检查输入数据格式
2. 降低增强强度
3. 减少同时启用的方法数量
4. 查看错误日志

### Q3: 如何批量导出数据？

**A**:
1. 在任务列表中勾选多个任务
2. 点击"批量操作"
3. 选择"导出"
4. 选择导出格式
5. 下载压缩包

### Q4: 支持自定义模型吗？

**A**:
- 当前版本提供 6 个预训练模型
- 未来版本将支持自定义模型上传
- 可以通过"模型配置"调整参数

### Q5: 数据安全如何保障？

**A**:
- 所有数据加密存储
- 访问权限控制
- 定期备份
- 安全审计日志

## 📈 性能优化建议

### 提高处理速度

1. **合理批次大小**
   - 图片：每批 100-500 张
   - 文本：每批 1000-5000 条

2. **选择合适模型**
   - 小数据集：轻量级模型
   - 大数据集：高性能模型

3. **优化增强配置**
   - 只启用必要的方法
   - 适当降低强度
   - 分批次处理

### 提高准确率

1. **数据质量**
   - 使用高质量训练数据
   - 清洗异常数据
   - 平衡数据分布

2. **模型选择**
   - 根据任务类型选择
   - 查看模型准确率
   - 测试多个模型

3. **参数调优**
   - 调整置信度阈值
   - 优化学习率
   - 增加训练轮次

## 🎓 最佳实践

### ✅ 推荐做法

1. **开始前规划**
   - 明确标注目标
   - 选择合适模板
   - 准备数据格式

2. **分批次处理**
   - 小批量测试
   - 验证结果
   - 逐步扩大

3. **定期备份**
   - 导出结果
   - 保存配置
   - 记录参数

4. **持续优化**
   - 分析准确率
   - 调整参数
   - 迭代改进

### ❌ 避免做法

1. **不要一次处理过多数据**
   - 可能导致超时
   - 难以调试错误

2. **不要跳过测试阶段**
   - 先小规模验证
   - 确认效果再扩大

3. **不要忽视数据质量**
   - 垃圾数据产生垃圾结果
   - 预处理很重要
