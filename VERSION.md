## 升级指南

### 从 v1.0.0 升级到 v2.0.0

#### 1. 更新依赖
```bash
pnpm install
```

#### 2. 清除缓存
```bash
rm -rf node_modules/.vite
```

#### 3. 重新构建
```bash
pnpm build
```

#### 4. 注意事项
- Logo 组件 API 已更新
- 新增动画类需要更新样式
- 键盘快捷键系统为新增功能
