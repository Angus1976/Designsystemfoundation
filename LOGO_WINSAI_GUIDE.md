## ✨ 最终设计理念

### "Wi" 笔触形的构成

```
视觉呈现：

         ●  ← 红点 #FF4D4F
         ｜ ← 第4笔 - i的主体（垂直）
        ╱ ╲  
       ╱   ╲ ← 第3笔 - W的右侧
      ╱  V  ╲
     ╱   ╲  ╲ ← 第2笔 - W的中间（V形）
    ╱     ╲  ╲
   ╱       ╲← 第1笔 - W的左侧
  ────────────────
      "Wi"
  
  WinsAI = Wi
  问视间 = 问(W)视(i)间
```

### 5个核心元素

| 元素 | 说明 | 象征意义 |
|------|------|----------|
| **第1笔** | 左下斜线 | 数据采集的起点 |
| **第2笔** | V形谷底 | 数据清洗处理 |
| **第3笔** | 右上斜线 | 数据上升标注 |
| **第4笔** | 垂直线 | AI智能训练 |
| **红点** | ● | SuperInsight洞察 ⭐ |

## 💡 快速使用

### HTML

```html
<!-- 推荐：静态版（日常使用） -->
<img src="/logos/logo-full-static.svg" alt="问视间 WinsAI" height="60">

<!-- 动画版（启动页、加载页） -->
<img src="/logos/logo-full.svg" alt="WinsAI SuperInsight" height="60">

<!-- Favicon -->
<link rel="icon" type="image/svg+xml" href="/logos/logo-favicon.svg">

<!-- Meta Tags -->
<meta property="og:image" content="https://www.wenshijian.com/logos/logo-square-256.svg">
<meta name="description" content="问视间 - 数据友好，智赢未来">
```

### React/TypeScript

```tsx
// Logo 组件
interface LogoProps {
  variant?: 'full' | 'full-static' | 'icon' | 'simple';
  animated?: boolean;
  size?: number;
}

export function WinsAILogo({ 
  variant = 'full-static', 
  animated = false,
  size = 60 
}: LogoProps) {
  const logoSrc = animated 
    ? '/logos/logo-full.svg'
    : `/logos/logo-${variant}.svg`;

  return (
    <img 
      src={logoSrc} 
      alt="问视间 WinsAI - 数据友好·智赢未来" 
      height={size}
      className="winsai-logo"
    />
  );
}

// 使用示例
<WinsAILogo variant="full-static" size={60} /> {/* Header */}
<WinsAILogo variant="icon" size={48} /> {/* 侧边栏 */}
<WinsAILogo animated={true} /> {/* 启动页 */}
```

## 📐 设计规范

### 核心规范

| 规范项 | 要求 | 说明 |
|--------|------|------|
| **最小尺寸** | 完整版≥180px，图标≥32px | 保证清晰度 |
| **安全区域** | Logo高度的20% | 周围留白 |
| **颜色保真** | 严格按照色值 | 特别是红点 #FF4D4F |
| **宽高比** | 保持原始比例 | 不可拉伸变形 |

### 禁止操作 ❌

1. ❌ **改变颜色**（特别是红点）
2. ❌ **拉伸变形**
3. ❌ **旋转Logo**
4. ❌ **移除红点**（这是核心辨识元素）
5. ❌ **分离笔画**
6. ❌ **改变笔画粗细**

### 推荐操作 ✅

1. ✅ 等比例缩放
2. ✅ 使用纯色背景
3. ✅ 保持清晰可辨
4. ✅ 确保红点醒目
