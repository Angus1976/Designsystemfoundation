## 🎓 使用指南

### 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/superinsight/superinsight.git
   cd superinsight
   ```

2. **安装依赖**
   ```bash
   pnpm install
   ```

3. **启动开发**
   ```bash
   pnpm dev
   ```

   ```
   http://localhost:5173
   ```
