### React/TypeScript

```tsx
// WinsAI Logo 组件
interface WinsAILogoProps {
  variant?: 'full' | 'full-static' | 'icon-64' | 'icon-128' | 'simple-48' | 'horizontal' | 'square-256';
  animated?: boolean;
  size?: number;
  className?: string;
}

export function WinsAILogo({ 
  variant = 'full-static', 
  animated = false,
  size = 60,
  className = ''
}: WinsAILogoProps) {
  const logoSrc = animated && variant === 'full'
    ? '/logos/logo-full.svg'
    : `/logos/logo-${variant}.svg`;

  return (
    <img 
      src={logoSrc} 
      alt="问视间 WinsAI - 数据友好·智赢未来" 
      height={size}
      className={`winsai-logo ${className}`}
      loading="lazy"
    />
  );
}

// 使用示例
// 官网Header
<WinsAILogo variant="full-static" size={60} />

// 仪表盘侧边栏
<WinsAILogo variant="icon-64" size={48} />

// 启动页（带动画）
<WinsAILogo variant="full" animated={true} size={80} />

// 移动端
<WinsAILogo variant="simple-48" size={36} />
```

## 📐 设计规范

### 核心规范

| 规范项 | 要求 | 说明 |
|--------|------|------|
| **最小尺寸** | 完整版≥180px，图标≥32px | 保证立体效果清晰 |
| **安全区域** | Logo高度的20% | 周围留白 |
| **颜色保真** | 严格按照深蓝色值 | 特别是 #0B2847 和红点 |
| **宽高比** | 保持原始比例 | 不可拉伸变形 |
| **立体效果** | 保持3面结构 | 不可简化为2D |

### 禁止操作 ❌

1. ❌ **改变深蓝配色**（这是专业形象核心）
2. ❌ **移除立体效果**（变成扁平设计）
3. ❌ **改变红点颜色**（核心辨识元素）
4. ❌ **拉伸变形**（破坏比例）
5. ❌ **旋转Logo**
6. ❌ **分离柱体**
7. ❌ **改变柱体高低关系**

### 推荐操作 ✅

1. ✅ 等比例缩放
2. ✅ 使用白色或浅色背景
3. ✅ 保持立体清晰
4. ✅ 确保红点醒目
5. ✅ 静态版用于日常（90%场景）
6. ✅ 动画版用于特殊场景
