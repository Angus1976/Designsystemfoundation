# 问视间 (SuperInsight) Logo 设计说明

## Logo 设计理念

### 核心概念
**问视间 (Wins)** - "通过问询获取洞察，在数据标注中赢得先机"

Logo 设计融合了以下核心元素：
- **数据流动**: 顶部波浪线代表数据的流动和处理过程
- **W 字形**: 中央的 "W" 造型象征 "Wins"（问视间），也代表数据的多维度分析
- **标注标识**: 右下角的绿色圆点象征数据标注的完成状态
- **底部线条**: 代表数据整理和结构化的基础

### 视觉元素

#### 1. 颜色方案
- **主渐变色**: #1890FF → #722ED1
  - 蓝色 (#1890FF): 代表科技、数据、专业
  - 紫色 (#722ED1): 代表 AI、智能、创新
  
- **标注指示色**: #52C41A (绿色)
  - 代表完成、准确、质量保证

- **白色元素**: 
  - 在深色背景上保持高对比度和清晰度

#### 2. 几何设计
- **圆形基础**: 32x32px 圆形或圆角矩形背景
- **描边宽度**: 2-2.5px，确保在各种尺寸下清晰可见
- **图标网格**: 基于 8px 网格系统设计

### Logo 变体

#### Full Logo（完整版）
- 包含图标 + "问视间" + "SuperInsight"
- 用于侧边栏展开状态、登录页面
- 尺寸：灵活适配容器

#### Icon Logo（图标版）
- 仅包含核心图标元素
- 用于侧边栏收起状态、favicon、移动端
- 标准尺寸：32x32px, 64x64px

#### Simple Icon（简化图标）
- 去除细节，保留核心 "W" 形状
- 用于极小尺寸显示
- 尺寸：16x16px, 24x24px

## 使用场景

### 1. 应用内使用
- **侧边栏**: 展开状态显示完整 Logo，收起状态显示简化图标
- **登录页面**: 使用放大的图标版本
- **移动端**: 使用简化图标以节省空间
- **加载状态**: 可使用 Logo 作为 loading 动画

### 2. 品牌应用
- **文档头部**: 使用完整 Logo
- **邮件签名**: 使用完整 Logo
- **社交媒体**: 使用图标版作为头像
- **网站 Favicon**: 使用简化图标

## 主题适配

### 明亮模式
- 背景保持渐变色
- 图标元素使用白色
- 标注点使用绿色

### 睿黑模式
- 背景保持渐变色（自动适配深色环境）
- 图标元素使用白色（高对比度）
- 标注点颜色稍微调整亮度以适配暗色主题

## 技术实现

### React 组件
```tsx
import { Logo } from '@/components/ui/logo';

// 完整版本
<Logo variant="full" showSubtitle={true} />

// 仅图标
<Logo variant="icon" />

// 简化图标
<LogoIconSimple className="w-8 h-8" />
```

- 组件: `/src/app/components/ui/logo.tsx`
- 使用位置: 
  - `/src/app/components/layout/AppLayout.tsx`
  - `/src/app/pages/Login.tsx`

## 设计原则

1. **简洁性**: Logo 设计简洁明了，易于识别和记忆
2. **可扩展性**: 在各种尺寸下都能保持清晰度和辨识度
3. **一致性**: 在不同主题和场景下保持视觉一致性
4. **语义化**: 设计元素与产品功能紧密关联
5. **现代感**: 采用渐变色和几何图形，体现科技感和现代商务风格
