## 🎨 Logo 变体说明

### 完整版 (Full Logo)
```tsx
<Logo variant="full" showSubtitle={true} />
```
- **尺寸**: 自适应容器
- **包含**: 图标 + "问视间" + "SuperInsight"
- **使用场景**: 侧边栏展开、品牌展示

### 图标版 (Icon Only)
```tsx
<Logo variant="icon" />
```
- **尺寸**: 32x32px (可缩放)
- **形状**: 圆形背景
- **使用场景**: 登录页面、应用图标

### 简化图标 (Simple Icon)
```tsx
<LogoIconSimple className="w-8 h-8" />
```
- **尺寸**: 32x32px (可缩放)
- **形状**: 圆角矩形背景
- **使用场景**: 收起的侧边栏、favicon

## 🔧 使用示例

### 基础用法
```tsx
import { Logo, LogoIconSimple } from '@/components/ui/logo';

// 在侧边栏中使用
function Sidebar() {
  return (
    <div>
      {expanded ? (
        <Logo variant="full" showSubtitle={true} />
      ) : (
        <LogoIconSimple className="w-8 h-8" />
      )}
    </div>
  );
}

// 在登录页面中使用
function LoginPage() {
  return (
    <div className="flex flex-col items-center">
      <div className="scale-125">
        <Logo variant="icon" />
      </div>
      <h1>问视间</h1>
      <p>SuperInsight</p>
    </div>
  );
}
```

### 自定义尺寸
```tsx
// 小尺寸
<Logo variant="icon" className="w-6 h-6" />

// 中等尺寸
<Logo variant="icon" className="w-8 h-8" />

// 大尺寸
<Logo variant="icon" className="w-12 h-12" />

// 使用 scale
<div className="scale-150">
  <Logo variant="icon" />
</div>
```
